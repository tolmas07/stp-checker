# Проверка по СТП 01-2024 БГУИР

Веб-приложение для автоматической проверки пояснительной записки дипломного проекта на соответствие требованиям **СТП 01-2024 БГУИР**.

## 🚀 Онлайн-версия

**[Открыть приложение →](https://tolmas07.github.io/stp-checker/)**

## Возможности

- ✅ Проверка полей страницы, шрифта, межстрочного интервала, отступов
- ✅ Проверка структуры документа (РЕФЕРАТ, СОДЕРЖАНИЕ, ВВЕДЕНИЕ, ЗАКЛЮЧЕНИЕ...)
- ✅ Проверка заголовков разделов и подразделов (нумерация, жирность, регистр, выравнивание)
- ✅ Проверка подписей рисунков и таблиц
- ✅ Проверка математических формул
- ✅ Проверка списка использованных источников
- ✅ Проверка знаков препинания, кавычек, тире, отступов
- ✅ Предварительный просмотр документа постранично
- ✅ Навигация к странице с ошибкой
- ✅ ИИ-анализ (Gemini / DeepSeek / OpenAI)

## Использование

1. Откройте [онлайн-версию](https://tolmas07.github.io/stp-checker/)
2. Перетащите файл `.docx` в область загрузки
3. Нажмите **«Проверить»**
4. Изучите отчёт об ошибках

## Локальный запуск

```bash
cd app
python -m http.server 8080
```
Откройте http://localhost:8080

## Технологии

- Vanilla JS / HTML5 / CSS3
- [JSZip](https://stuk.github.io/jszip/) — разархивирование .docx
- [Mammoth.js](https://github.com/mwilliamson/mammoth.js/) — конвертация в HTML для предпросмотра
- GitHub Pages — хостинг
